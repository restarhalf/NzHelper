## 简介

DeerTimer 是一个基于 Jetpack Compose 开发的轻量记录工具，用来记录每次“练习”的时长与备注，并提供统计分析、历史记录、排行榜与云同步等功能。

## 功能特性

- **计时**
    - 开始 / 暂停 / 结束
    - 结束后可填写本次记录的详细信息
- **记录**
    - 备注 / 地点 / 道具 / 心情 / 评分等
- **历史**
    - 查看详情、编辑、删除、清空
    - 导入 / 导出
- **统计**
    - 周 / 月 / 年统计
    - 图表展示 + 总体数据汇总
- **主题与外观**
    - 主题色模式切换
    - 底栏固定 / 悬浮
    - 背景图、背景透明度、背景模糊、组件透明度
- **账号与云功能**
    - 登录 / 注册 / 个人资料
    - 云同步
    - 排行榜
